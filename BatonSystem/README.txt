#Baton System

Baton System is a Java/ JAS-RX implemented application for dealing with trade

# Software

This application uses STS IDE (JRE 1.8) and JAX-RX packages to implement

#Application Deployment and Testing

Application deployed in Tomcat 9.0 and tested on Talend API tester

# Others

This application uses in memory databse (Java Map) which is not for production envioument at all.

All funtionalities are not implemented due to shortage of time.

Codes can be modified for better performance

For Running the application 
http://Your Port:serverport/BatonSystem/webapi/trade
http://Your Port:serverport/BatonSystem/webapi/trade/INFY
http://Your Port:serverport/BatonSystem/webapi/trade/IBM
etc
